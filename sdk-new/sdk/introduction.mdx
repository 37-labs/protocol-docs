The Sage SDK makes it easy to add decentralized social features to your applications. Built specifically for Sage Protocol, the SDK handles complex blockchain tasks behind the scenes, simplifying how you manage user identities, social interactions, content, and rewards.

## Main Features

The SDK provides straightforward APIs and components to:

- **Easy Content Creation**: Quickly create and interact with blockchain-based posts, likes, and comments.
- **Decentralized Identities**: Effortlessly manage user profiles, relationships, and preferences onchain.

- **Community Tools**: Integrated moderation controls, customizable community management, and options to sponsor user transaction costs.

Leverage Sage SDK to build:
