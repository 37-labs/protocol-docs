Install the SDK:

<CodeGroup>

```bash pnpm
pnpm i @sageprotocol/sdk
```


```bash yarn
yarn add @sageprotocol/sdk
```


```bash npm
npm i @sageprotocol/sdk
```

</CodeGroup>

The Sage SDK supports two integration methods:

- **Direct JavaScript Integration:** Offers direct use of the SDK through the `SageClient`.
- **React Integration:** Uses built-in React hooks and the `SageProvider` component for easy state management.

<Info>
  Both approaches offer similar capabilities:

  - Use the same transaction result pattern (`{ ok, err?, transaction? }`)
  - Require manual wallet integration for transaction signing
  - Support the same networks (testnet/mainnet)
  - Provide identical error handling and validation
  - Offer the same performance characteristics

  The React wrapper is simply a convenience layer built on top of the vanilla client.
</Info>
