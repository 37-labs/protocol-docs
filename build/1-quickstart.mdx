## Prerequisites

Before you start, ensure you have:

<Check>
  ·       **Sui Wallet:** Installed and configured (Sui Wallet Guide).
</Check>

<Check>
  ·       **Node.js (v18 or later):** Installed ([Download Node.js](https://nodejs.org/)).
</Check>

<Check>
  ·       **Sage SDK:** Sage Protocol's official client library (Available here).
</Check>

## Setup and Installation

Set up your development environment quickly with these steps, depending on your tech stack:

<Steps>
  <Step title="Install Sage SDK">
    ```
    npm install @sageprotocol
    ```
  </Step>
  <Step title="Configure Sage React Provider or Client">
    ```
    import { SageProvider } from '@sageprotocol/react';
    
    return (
        <SageProvider appId=”your-app-id” network=”testnet”>
            …
        </SageProvider>
    );
    
    or
    
    import { SageClient } from '@sageprotocol/sdk';
    
    const sage = new SageClient({
      appId: 'your-app-id',
      network: 'testnet', // or 'mainnet'
    });
    ```
  </Step>
</Steps>

Ensure your Sui wallet is connected to the appropriate network (Sui Testnet or Sui Mainnet).
