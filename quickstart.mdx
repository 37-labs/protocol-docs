## 2.  Setup and Installation

Set up your development environment quickly with these steps:

### 1.  Install Sage SDK

```
npm install @sageprotocol/sdk
```

### 2.  Configure Sage Client:

```
import { SageClient } from '@sageprotocol/sdk';

const sage = new SageClient({
  appId: 'your-app-id',
  network: 'testnet', // or 'mainnet'
});
```

Ensure your Sui wallet is connected to the appropriate network (testnet or mainnet).
